# Naming Rules

## Naming Protection / 네이밍 불변 규칙

### 1. 기존 이름 변경 금지

- 함수 / 변수 / 클래스 / 파일 / API / DB 컬럼

### 2. 자동 변경 금지

- 리네이밍
- casing 변경
- 축약/확장

### 3. 변경 감지

- 이름 변경 감지 시 즉시 중단

### 4. 허용 조건

- 명시적 요청 + 테스트 통과

### 5. 작업 방식

- 기존 이름 재사용

### 6. 실패 처리

- 롤백 + 중단

### 7. DB 보호

- 컬럼/타입 변경 금지

---

## 네이밍 금지 행동

- 의미 추측 변경
- snake ↔ camel 변환
- 축약 생성
- 복수/단수 변경
